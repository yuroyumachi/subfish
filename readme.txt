Subfish 语言

Subfish 是一个极简主义语言
其基于三个无限精度的累加器, 通过简洁的指令集实现计算.

概述
* 累加器 -三个整数寄存器, [a0, a1, a2], 初始值均为 0
* 当前累加器 -一个指针, 指向三个累加器之一, 初始值为 0
* 代码指针 -指向当前正在执行的指令字符
* 标记指针 -用于保存代码指针

可用指令集
* i -当前累加器加 1
* d -当前累加器减 1
* s -当前累加器取平方
* o -打印当前累加器的十进制值, 换行
* c -将当前累加器的值是为 Unicode 码点,输出对应字符, 不换行
* k -从标准输入读取一个字符, 将其 Unicode 码点存入当前累加器, 如果读取到文件末尾, 则不做任何修改
* h -立刻终止程序, 如果可以返回 0
* t -切换当前累加器(a0 -> a1 -> a2)
* z -如果当前累加器的值为 0, 跳过 z 往后的 10 个字符(不包括 z)
* m -保存当前代码指针到标记指针
* p -将代码流跳转到标记指针所指的位置
* n -什么也不做, 用于填充 z 跳过的操作符的控件

预处理
在程序开始执行之前, 会先开始预处理
预处理包括以下步骤
1. 去除所有被 # 之后以及换行之前的字符
2. 去除所有不在可用指令集之内的字符
